# **Base de datos para el componenete lógico de gestión re reclamos**
Su responsabilidad es alemcenar la información relevante que se genera en la gestión integral del reclamo. Además, de los estados del reclamo y los identificadores de documentos.
En lo referente a su compocisión; es una base de datos NoSQL que permite una organización por colecciones, lo cual permite una estructura flexible y escalable. Es un servicio ideal para la lógica de negocio que rige este componente.
